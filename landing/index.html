The problem

Your AI has amnesia

Every session starts from zero. Your AI doesn't remember the architecture decisions from last week, the debugging approach that worked, or that you prefer tabs over spaces.

You've explained the same context dozens of times. Your agent asks the same questions. Makes the same mistakes. Suggests patterns you've already rejected.

The fix

Install. Index. Remember.

Install BrainLayer, add it to your MCP config, and index your conversations. Your agent now searches 284K chunks of knowledge before responding — past decisions, file histories, entity relationships, your corrections.

Hybrid search (semantic + keyword + knowledge graph) finds the right memory in <50ms. Zero cloud. Everything stays on your machine.

claude code
# Your agent now remembers everything you: What approach did we use for auth? brain_search("auth approach architecture") Found 3 chunks (12ms) → "Chose JWT with refresh tokens over session cookies. Reason: stateless API, mobile client support. Decision date: 2026-02-14. Confidence: high." claude: You chose JWT with refresh tokens on Feb 14. The rationale was stateless APIs and mobile support. How it works

Memory that actually works

🔍

Hybrid search

Semantic vectors, FTS5 keywords, and knowledge graph — merged with Reciprocal Rank Fusion. The right memory in <50ms.

⚡

Real-time indexing

JSONL watcher indexes your conversations within seconds. 4-layer content filters keep noise out and signal in.

🏷️

10-field enrichment

Every chunk gets summary, tags, importance, intent, symbols, epistemic level, and more. Via Groq or local LLM.

🕸️

Knowledge graph

Entities, relations, evidence chains. Ask "what do we know about this person?" and get a structured answer, not a keyword match.

♻️

Chunk lifecycle

Supersede, archive, filter. Stale knowledge is managed, not lost. Entity contracts with health scoring keep your graph clean.

🖥️

BrainBar daemon

209KB native macOS Swift binary. Always-on MCP access over Unix socket. Starts on login, invisible until needed.

MCP Interface

11 tools. One protocol.
